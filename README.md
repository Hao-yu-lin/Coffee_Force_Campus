# ☕ Coffee Force Campus - Brewing Analysis

這是一個用於咖啡沖煮數據分析與 CVA (Coffee Value Assessment) 評測的網頁應用程式。提供圖表視覺化、多重資料集比對，以及完整的 CVA 品鑑表單（支援電腦與手機介面）。

## 🚀 執行步驟

為了確保所有功能（如圖表繪製、檔案匯入等）能正常運作且不受瀏覽器的安全性限制 (CORS)，請透過終端機建立本機伺服器 (Local Web Server) 來執行此專案。

### 步驟一：開啟終端機並建立連線

1. 開啟終端機 (Terminal / 命令提示字元)。
2. 將目錄切換到本專案的資料夾底下：
   ```bash
   cd /你的/專案/路徑/Coffee_Force_Campus
   ```
3. 啟動本機伺服器（推薦使用 Python 內建的伺服器）：
   ```bash
   python -m http.server 8000
   ```
   *(伺服器啟動後，請保持終端機視窗開啟，不要關閉)*

---

### 步驟二：在電腦 (PC) 上開啟

如果是使用電腦操作，請直接打開瀏覽器（推薦使用 Chrome 或 Edge），在網址列輸入以下網址即可進入系統：
```text
http://localhost:8000/main.html
```

---

### 步驟三：在手機 (Mobile) 上開啟

若想在手機上進行操作，請確保 **「手機」與「電腦」連線至同一個 Wi-Fi 網路**，然後按照以下步驟：

1. **確認電腦的 IP 位址：**
   - **Windows系統：** 另外開啟一個終端機輸入 `ipconfig`，找到「IPv4 位址」 (例如：`192.168.1.100`)。
   - **Mac/Linux系統：** 另外開啟一個終端機輸入 `ifconfig` 或 `ipconfig getifaddr en0`，尋找目前的本機 IP。
2. **組合 IP 位址：** 將查到的 IP 位址與剛剛開啟的 Port (8000) 結合，並加上 `http://`。例如：`http://192.168.1.100:8000/main.html`
3. **手機開啟：** 打開手機的瀏覽器，輸入上面組合好的網址，就可以看到針對手機版最佳化過的操作介面了！
